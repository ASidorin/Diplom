# Общий итог

### Что было запланировано и что было сделано

Разработанный нами план тестирования был реализован полностью.
Нами был создан проект тестирования на основе java 11 и Gradle.
Были собраны все необходимые для этого зависимости. 
Автоматизация тестов была проведена на основе Selenide.
Были подключенны сторонние сервисы через Docker containers для генерации тестируемых
данных написанных на основе Node.js.
Были подключены и проверена работа двух баз данных Mysql и Postgesgl.
Удалось подключить CI-CD appveyor.
Отчеты выводились через Allure-report.
